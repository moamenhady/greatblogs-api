Welcome to GreatBlogs API

About

This is an Open Source Spring Boot API providing access to a blog system, allowing users to signup and perform various CRUD operations related to blog posts. The API is running as a Docker container on Azure Container Apps. The database used is PostgreSQL running on Azure Database for PostgreSQL - Flexible Server.
