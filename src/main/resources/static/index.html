Welcome to GreatBlogs API

About

This is an Open Source Spring Boot API providing access to an authenticated blog system, allowing users to signup and perform various CRUD operations related to blog posts.

The API is running as a Docker container on Azure Container Apps. The database used is PostgreSQL running on Azure Database for PostgreSQL - Flexible Server.

Usage

  • POST https://greatblogs.tech/api/authors/signup

    Sign-up a new author account with username, password and email.


  • GET https://greatblogs.tech/api/authors/all

    Retrieve all authors by username.


  • GET https://greatblogs.tech/api/authors/{id}

    Retrieve a specific author by id.


  • PUT https://greatblogs.tech/api/authors/update/{id}

    Update an existing author's fullName and about.


  • POST https://greatblogs.tech/api/posts/create

    Create a new blog post.


  • GET https://greatblogs.tech/api/posts/all

    Retrieve all blog posts.


  • GET https://greatblogs.tech/api/posts/{id}

    Retrieve a specific blog post by id.


  • PUT https://greatblogs.tech/api/posts/update/{id}

    Update an existing blog post.


  • DELETE https://greatblogs.tech/api/posts//delete/{id}

    Delete a blog post.
